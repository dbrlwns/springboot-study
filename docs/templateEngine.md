# Thymeleaf

### 타임리프 표현식
|표현식| 설명                                  |
|---|-------------------------------------|
|${...}| 변수의 값 표현식                           |
|#{...}| 속성 파일 값 표현식                         |
|@{...}| URL 표현식                             |
|*{...}| 선택한 변수의 표현식, th:object에서 선택한 객체에 접근 |


### 타임리프 문법
|표현식| 설명 |예제|
|---|----|---|
|th:text|텍스트를 표현할 때 사용|th:text=${person.name}|
|th:each|컬렉션을 반복할 때 사용|th:each="person: ${persons}"|
|th:if|조건이 true인 때만 표시|th:if="${person.age} >= 20"|
